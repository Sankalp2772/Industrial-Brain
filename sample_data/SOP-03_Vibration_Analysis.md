# Standard Operating Procedure: SOP-03
**Title:** Vibration Analysis Standards
**Last Updated:** 2023-01-15
**Author:** Alex Chen

## 1. Thresholds for Rotating Machinery
- **Normal:** < 0.15 in/s
- **Warning (Alert):** 0.15 - 0.25 in/s
- **Critical (Danger):** > 0.25 in/s

If vibration exceeds the Critical threshold, the asset MUST be shut down immediately to prevent structural damage.
